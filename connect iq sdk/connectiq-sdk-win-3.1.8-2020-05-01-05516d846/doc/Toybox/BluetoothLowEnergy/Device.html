Overview

Represents a Bluetooth Low Energy Device that has been paired with the system.

This class cannot be instantiated, access to paired system devices is done through the use of Toybox::BluetoothLowEnergy#getPairedDevices() or by calling Toybox::BluetoothLowEnergy#pairDevice() after receiving a ScanResult for a device
